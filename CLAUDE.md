# Git Commit Convention

커밋 시 반드시 아래 형식을 준수한다.

**형식:** `이모지 타입: 작업 내용 요약`

| 이모지 | 타입 | 설명 |
| :--- | :--- | :--- |
| ✨ | Feat | 새로운 기능 추가 |
| 🐛 | Fix | 버그 수정 |
| ♻️ | Refactor | 코드 구조 개선 |
| 🔧 | Chore | 빌드 설정, 의존성 수정 |
| 📝 | Docs | 문서 수정 |

예: `✨ Feat: 회원 가입 기능 구현`
예: `♻️ Refactor: 패키지 구조 도메인별로 전환`
